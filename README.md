# Food Delivery Backend System

A backend application for a food delivery platform developed using Java, Spring Boot, and MySQL. The project follows layered architecture principles and provides RESTful APIs for managing users and backend operations.

---

## Features

- User Registration API
- User Management
- REST API Development
- DTO-Based Request Handling
- Global Exception Handling
- Layered Architecture
- MySQL Database Integration

---

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman

---
